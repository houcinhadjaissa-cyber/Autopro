# Autopro

Autopro is a multi-layer e-commerce website for selling car parts online.

## Goal

Build a full car parts e-commerce platform that includes:

- Customer storefront
- Product catalog
- Vehicle-based part search
- Shopping cart
- Checkout
- User accounts
- Admin dashboard
- Inventory management
- Order management
- Backend API
- Database

## Project status

We are at the beginning of the project.

Current step:

- Create the GitHub repository
- Add the first project files
- Plan the structure before writing code

## Planned layers

### 1. Storefront

The public website where customers can:

- Browse car parts
- Search parts
- Filter by vehicle, brand, category, and price
- View product details
- Add items to cart
- Checkout

### 2. Admin Dashboard

The private admin area where the store owner can:

- Add products
- Edit products
- Manage inventory
- View orders
- Manage customers
- Update categories

### 3. Backend API

The server layer that handles:

- Products
- Users
- Authentication
- Cart
- Orders
- Payments
- Admin actions

### 4. Database

The database will store:

- Products
- Categories
- Car makes and models
- Users
- Orders
- Inventory
- Payments

## Important project rule

We will save progress step by step using GitHub commits so we do not lose work.
