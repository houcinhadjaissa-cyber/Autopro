# Autopro Platform Blueprint

This file describes the intended backbone of the Autopro platform.

Autopro is not only a basic online store. The long-term vision is to build a car parts e-commerce platform with marketplace features and a vehicle compatibility engine.

---

# Project Reality and Constraints

The project owner currently has:

- No laptop
- Limited budget
- 0% coding experience
- Mobile-only workflow at the beginning

Because of this, the project must be built carefully with:

- Simple explanations
- Step-by-step progress
- GitHub saves/commits after important changes
- No rushed coding
- Clear documentation
- A first simple version before advanced features
- A structure that can grow later
