# Homepage & Visual Experience Requirements

## 1. The "First Impression" Hero Section
*   **3D Interactive Background:** The top of the homepage must feature a 3D car model. As the user scrolls or interacts, the car "disassembles" (explodes view) with a 360° camera rotation to show internal parts.
*   **Primary Navigation Tabs:** Three intuitive, high-visibility tabs over the 3D car:
    1.  **My Garage** (Garage Icon): Quick access to saved vehicles.
    2.  **Business** (Shop/Store Icon): For suppliers and service providers.
*   **Global Search Bar:** Positioned prominently underneath the tabs, expanding into a smart selector/filter for parts and services.

## 2. Visual Parts Catalog (7zap Style)
*   **Brand Grid:** A row of professional brand logos (Toyota, Mercedes, Hyundai, etc.).
*   **Drill-Down Logic:** Clicking a logo must expand into a clean, white-background visual grid showing:
    1.  **Models:** All available models for that brand.
    2.  **Generations:** Specific generations (e.g., W211, W212) including clear distinctions for **facelifts**.
*   **Visual Precision:** Every model and generation must use a high-quality, professional white-background image to match the 7zap aesthetic.

## 3. Localization & Automation
*   **Auto-Detection:** The system must automatically detect the user's country (Algeria/Nigeria) to show local currency (DZD/NGN) and local language preferences immediately.
*   **Dynamic Rows:** The homepage will include automated sections:
    1.  **Trending Spare Parts** (Based on popular searches).
    2.  **Top Rated Mechanics/Garages** (Based on trust scores).
    3.  **Video Discovery** (TikTok-style short video feed).
    4.  **Discounted Deals** (At the bottom of the page).

## 4. Design Standards
*   **Cross-Platform UI:** Use the best UX elements from Facebook (navigation), Instagram (visuals), TikTok (video feed), and Amazon (checkout).
*   **Return User Shortcut:** Returning users must see an instant "Find parts for my [Saved Car]" button to bypass manual searching.
