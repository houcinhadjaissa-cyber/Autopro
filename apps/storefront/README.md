# Autopro Storefront

This folder will contain the customer-facing website.

The storefront is where customers will:

- Browse car parts
- Search by vehicle
- View product details
- Add products to cart
- Checkout
- Create accounts
- View order history

## Planned pages

- Home page
- Product catalog page
- Product details page
- Search results page
- Cart page
- Checkout page
- Login page
- Register page
- Customer account page
